# AR Tech Solutions - Integrated ERP, CRM, & HRM

This repository contains the white-labeled ERPNext v16 instance for **AR Tech Solutions**. It includes custom branding, integrated HRM (Version-16), and CRM modules.

## 🚀 Quick Start

### 1. Prerequisites
- Docker and Docker Compose installed on your system.

### 2. Download and Setup
Clone this repository to your local machine:
```bash
git clone https://github.com/Artechsolutions-arts/erp_crm_hrms.git
cd erp_crm_hrms
```

### 3. Launch the Application
Run the following command to start all services:
```bash
docker compose -f "pwd (1).yml" up -d
```

### 4. Access the System
Once the containers are running, open your browser and go to:
**http://localhost:8000**

---

## 📝 Maintenance
To update the system or restart services:
```bash
docker compose -f "pwd (1).yml" restart
```
